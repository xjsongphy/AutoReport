# 报告写作规范与指南

本文档整合了北京大学近代物理实验课程的官方要求和 AutoReport 的补充叙述风格指南。

## 官方模板说明

**主模板**: `template.tex` - 北京大学物理学院近代物理实验报告模板（基于 PKUMpLtX 文档类）

该模板综合了 American Institute of Physics (AIP) 期刊和中国《物理学报》对稿件格式的要求，包含完整的报告结构和详细的注释说明。

### 使用方法

1. **模板文件**: `template.tex` - 包含完整的报告结构和示例
2. **文档类**: `\documentclass[font=default]{mpltx}`
3. **编译方式**: `xelatex`（需两次编译以处理交叉引用）

```bash
xelatex -interaction=nonstopmode template.tex
xelatex -interaction=nonstopmode template.tex
```

## 报告结构

### 标准章节顺序

1. **标题页**
   - 实验题目：切合报告内容，简短明确
   - 作者、学号、联系方式
   - 摘要（200-300字）
   - 关键词（2-4个）

2. **引言**
   - 研究领域背景和现状
   - 有待研究的问题
   - 本研究的物理目的和主要方法
   - 篇幅不超过报告文字的1/3

3. **理论**（可选）
   - 概括本实验必须的理论
   - 给出必要的公式并编号
   - 公式应作为句子的一部分，末尾给出标点

4. **实验装置**
   - 实验方法和条件（成段介绍，非罗列步骤）
   - 实验装置示意图
   - 实验仪器和方法的介绍
   - 不确定度估计（简单可在此节，复杂的放到分析讨论）

5. **结果及讨论**（主体部分，占一半以上）
   - 实验结果以图表形式给出
   - 每个图表应完整，可独立理解
   - 以图表为中心叙述实验结果和讨论
   - 对预料之外的结果需先证明可靠性

6. **结论**
   - 给出实验结果
   - 给出由实验结果分析得到的结果和结论
   - 比摘要更全面，用词更准确

7. **致谢**（可选）
8. **参考文献**
9. **附录**：思考题

## 叙述风格指南（AutoReport 补充）

### 核心原则

1. **叙述先于内容**
   - 不要直接以列表、表格或公式开始章节
   - 始终先包含解释性文字
   - 叙述与内容交织（narrative-content-narrative）

2. **使用粗体而非斜体**
   - 用 **粗体** 表示强调
   - 技术报告中避免使用斜体

3. **完整的句子**
   - 每个句子语法完整
   - 表达一个清晰的思路
   - 避免句子片段

4. **避免对话式填充语**
   - 避免："我们将探索"、"正如我们所见"、"值得注意的是"
   - 保持直接和专业

5. **变量定义在公式之前**
   - 在写公式之前指定变量集合
   - 使用短语如"取任意..."、"对于..."、"令..."

### 示例

**不好的例子:**
```latex
\section{结果}

表1显示了数据。

我们可以看到：
- 数值是 9.81
- 不确定度是 0.02

方程是：
$$F = ma$$
```

**好的例子:**
```latex
\section{结果}

表1展示了通过自由落体实验测得的重力加速度数据。数据是使用毫秒级精度的数字计时器采集的。

测量值为 9.81 ± 0.02 m/s²，与理论预测值 9.81 m/s² 在实验不确定度范围内一致。这证实了实验装置准确地模拟了地球重力场中的自由落体运动。

对于质量为 $m$、受净力 $F$ 作用的物体，牛顿第二定律指出：

$$
F = ma
$$

其中 $a$ 是产生的加速度。对于地球表面附近的自由落体，$F = mg$，因此 $a = g$。
```

## 图表规范（基于顶级期刊要求）

### 图片要求

**技术规格:**
- **分辨率**:
  - 照片/灰度图: 300-600 DPI
  - 线图/图表: 600-1200 DPI（推荐 1000 DPI）
  - 组合图: 600 DPI
- **格式**: PNG（用于 LaTeX），TIFF/EPS（用于期刊投稿）
- **尺寸**:
  - 单栏: 80-89 mm（3.25-3.5 英寸）
  - 双栏: 170-180 mm（7 英寸）
  - 最大高度: 230 mm
- **字体**: Arial 或 Helvetica，8-12 point
- **面板标签**: 多面板图使用大写字母（A、B、C）

**科学标准:**
- 坐标轴标签附带单位（括号内）
- 存在不确定度时显示误差棒
- 多数据集时显示图例
- 清晰的标题描述所展示内容

**美学标准:**
- 使用对色盲友好的配色方案
- **避免红绿颜色组合**（非色盲友好）
- **避免彩虹色方案**（使用感知均匀的色彩映射，如 viridis、plasma）
- 确保灰度模式下仍可解释

**颜色最佳实践:**
- 使用色盲友好调色板（viridis、plasma、cividis）
- 考虑灰度兼容性
- 用颜色突出重点，而非装饰
- 在灰度模式下测试图表

### 表格要求

**结构:**
```latex
\begin{table}
  \caption{测量结果}
  \begin{ruledtabular}
    \begin{tabular}{cd{4.2}lr}
      居中    & 数值 & 靠左 & 靠右 \\
      \colrule
      a     & 0.12  & a   & a \\
      bbb   & -100  & bbb & bbb \\
    \end{tabular}
  \end{ruledtabular}
\end{table}
```

**指导原则:**
- 首尾使用双横线（`ruledtabular` 环境）
- 在列标题或数值中包含单位
- 使用适当的有效数字报告不确定度
- 避免竖线
- 保持一致的小数对齐

## 写作一般事项（官方要求）

1. 读者假定：既不是已知全部细节的指导教师，也不是缺少专业知识的公众，而是同领域的实验研究者
2. 文本和物理量单位用正体，物理变量符号用斜体，矢量矩阵符号用黑斜体
3. 缩略词首次出现时写出全称，后附缩略词并用括号括起；之后直接用缩略词
4. 全文标点符号除"顿号"外，其他用英文半角标点符号
5. 公式、图和表要分别用阿拉伯数字编列序号，达到可发表的质量
6. 凡不是自己独立思考得到的内容都应该引参考文献
7. 只能引用正式出版物，不能引用他人实验报告
8. 较长的推导和说明可以作为附件提交
9. 思考题作为附录附在报告最后

## 引用格式

### 正文引用

```latex
这是对实验教材的引用 \cite{jindaishiyan}。

引用标记要出现在标点之前 \cite{GBT7714,pr}。

连续引用也可以 \cite{PhysRevLett.116.061102}。
```

### 交叉引用

使用 `\autoref` 命令自动给出带类型的引用：

```latex
如 \autoref{sec:theory} 所示...

参见 \autoref{fig:instruments}...

从公式 \autoref{eq:1} 可以看出...

表 \autoref{tab:table_eg} 显示了...

见附录 \autoref{app:exercise}...
```

## 参考文献部分

```latex
\bibliography{bibli}
```

使用 BibTeX 处理参考文献，`*.bib` 文件可以从出版商网站、Google 学术、百度学术等获得。

## LaTeX 编译

### 编译命令

```bash
# 必须编译两次以处理交叉引用
xelatex -interaction=nonstopmode main.tex
xelatex -interaction=nonstopmode main.tex
```

### 为什么需要两次编译？

- 第一次编译：生成 `.aux` 文件，包含标签信息
- 第二次编译：解析交叉引用

## 不确定度分析指南

### 报告不确定度

- 所有数据都附带测量不确定度
- 通过计算传播误差
- 使用适当的有效数字
- 区分系统误差和随机误差

### 有效数字

- 最终结果舍入到适当的有效数字
- 中间计算应保留额外位数
- 不确定度通常报告1-2位有效数字

### 示例

```latex
测得的重力加速度值为
$g = 9.81 \pm 0.02$ m/s²。这与理论值
$9.81$ m/s² 在实验不确定度范围内一致，
证实实验装置准确模拟了地球重力场中的自由落体运动。
```

## 质量检查清单

提交报告前检查：

### 内容质量
- [ ] 所有章节以解释性文字开始
- [ ] 叙述流畅连贯（非内容堆砌）
- [ ] 所有变量在使用前定义
- [ ] 句子完整且语法正确
- [ ] 无对话式填充语
- [ ] 结果及讨论部分占一半以上篇幅

### 图表质量
- [ ] 所有图片正确标记和标注
- [ ] 分辨率符合要求（300-1200 DPI）
- [ ] 使用色盲友好配色
- [ ] 坐标轴附带单位
- [ ] 图表标题描述充分
- [ ] 多曲线在同一图上以便比较

### 表格质量
- [ ] 使用 `ruledtabular` 环境
- [ ] 首尾双横线
- [ ] 单位包含在标题或数值中
- [ ] 不确定度正确报告
- [ ] 小数点对齐

### 格式质量
- [ ] 所有引用正确解析
- [ ] 交叉引用有效
- [ ] 编译两次处理交叉引用
- [ ] PDF 成功生成
- [ ] 参考文献格式正确
- [ ] 缩略词首次出现时给出全称

### 官方要求
- [ ] 假定读者为同领域研究者
- [ ] 物理量单位用正体，变量用斜体
- [ ] 使用英文半角标点
- [ ] 公式图表编号
- [ ] 引用正式出版物
- [ ] 思考题在附录中
