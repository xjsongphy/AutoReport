# Report Requirements and Guidelines

This document specifies the default reporting requirements for physics experiment reports generated by AutoReport.

## Narrative Style Guidelines

### Core Principles

1. **Narrative Flow First**
   - NEVER start a section directly with a list, table, or formula
   - ALWAYS include explanatory text first
   - Weave narrative through content (narrative-content-narrative)

2. **No Italics, Use Bold**
   - Use **bold** for emphasis only
   - Never use italics in technical reports

3. **Complete Sentences**
   - Every sentence must be grammatically complete
   - Express one clear thought per sentence
   - Avoid sentence fragments

4. **No Conversational Filler**
   - Avoid: "we will explore", "as we can see", "it is interesting to note"
   - Be direct and professional

5. **Variable Domains Before Formulas**
   - Specify variable sets BEFORE writing formulas
   - Use phrases like "Take any...", "For...", "Let..."

### Examples

**BAD example:**
```
## Results

Table 1 shows the data.

We can see that:
- The value is 9.81
- The uncertainty is 0.02

The equation is:
$$F = ma$$
```

**GOOD example:**
```
## Results

Table 1 presents the measurements of the acceleration due to gravity obtained from the free-fall experiment. The data were collected using a digital timer with millisecond precision.

The measured value of 9.81 ± 0.02 m/s² agrees with the theoretical prediction of 9.81 m/s² within experimental uncertainty. This confirms that the experimental setup accurately models free-fall motion under Earth's gravitational field.

For any object with mass $m$ subject to a net force $F$, Newton's second law states:

$$
F = ma
$$

where $a$ is the resulting acceleration. In the case of free-fall near Earth's surface, $F = mg$ and therefore $a = g$.
```

## Figure and Table Conventions

### Figures

**Requirements:**
- Resolution: 300 dpi minimum
- Format: PNG with transparent background when appropriate
- Size: 6-8 inches wide for LaTeX document
- Fonts: Sans-serif at readable size

**Structure:**
```latex
\begin{figure}[h]
\centering
\includegraphics[width=0.8\textwidth]{figures/plot1.png}
\caption{Position vs time for falling object. The data points (blue) show the measured positions, while the red curve shows the theoretical prediction for constant acceleration.}
\label{fig:position}
\end{figure}
```

**Caption Guidelines:**
- Descriptive, explaining what the figure shows
- Interpret significance, not just describe
- Reference in main text before the figure

### Tables

**Structure:**
```latex
\begin{table}[h]
\centering
\caption{Measurement results}
\begin{tabular}{ccc}
\hline
Trial & Value (m/s²) & Uncertainty \\
\hline
1 & 9.81 & 0.02 \\
2 & 9.79 & 0.02 \\
3 & 9.82 & 0.02 \\
\hline
\end{tabular}
\end{table}
```

**Guidelines:**
- Use `booktabs` for professional formatting (`\toprule`, `\midrule`, `\bottomrule`)
- Include units in column headers or with values
- Report uncertainties with proper significant figures

## Citation Format

### In-Text Citations

```latex
According to Smith et al. (2020)...

The theoretical framework was established by Johnson (2019).

Multiple studies have confirmed this effect (Smith 2020; Johnson 2019; Lee 2021).
```


## Report Structure

### Standard Sections

1. **Title Page**
   - Experiment title
   - Author name
   - Date
   - Course/Instructor information

2. **Introduction**
   - Background and motivation
   - Objectives
   - Overview of approach

3. **Theory**
   - Fundamental principles
   - Derivations
   - Relevant equations

4. **Experimental Setup**
   - Apparatus description
   - Procedure
   - Measurements taken

5. **Data Analysis**
   - Methods used
   - Calculations performed
   - Error analysis

6. **Results**
   - Present findings
   - Tables and figures
   - Statistical analysis

7. **Discussion**
   - Interpret results
   - Compare with theory
   - Address discrepancies

8. **Conclusion**
   - Summary of findings
   - Implications
   - Suggestions for improvement

9. **References**
   - Cite all sources
   - Follow consistent format

## Error Analysis Guidelines

### Reporting Uncertainties

- Report measurement uncertainties with all data
- Propagate errors through calculations
- Use appropriate significant figures
- Distinguish between systematic and random errors

### Significant Figures

- Round final results to appropriate significant figures
- Intermediate calculations should keep extra digits
- Uncertainties typically reported with 1-2 significant figures

### Example

```latex
The measured value of the acceleration due to gravity is
$g = 9.81 \pm 0.02$ m/s². This agrees with the theoretical value of
$9.81$ m/s² within experimental uncertainty, confirming that the
experimental setup accurately models free-fall motion.
```

- [ ] Complete sentences throughout
- [ ] No conversational filler
- [ ] All figures properly labeled and captioned
- [ ] All tables properly formatted
- [ ] Cross-references resolve correctly
- [ ] Compiled twice for cross-references
- [ ] PDF generated successfully
- [ ] Narrative flows coherently between sections
- [ ] Errors and uncertainties properly reported
- [ ] References properly cited
